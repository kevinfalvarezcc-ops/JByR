💾 USB, Sistemas Operativos y Arranque

Aprende a crear USBs booteables, conoce las herramientas y cómo acceder a la BIOS.

¿Qué es un USB?

USB (Universal Serial Bus) es un tipo de almacenamiento portátil que se conecta al puerto USB de la computadora. Sirve para:

  • Almacenar y transferir archivos (documentos, fotos, videos).
  • Crear copias de seguridad.
  • Instalar sistemas operativos (USB booteable).
  • Ejecutar herramientas de diagnóstico o recuperación.
Capacidades típicas: desde 4 GB hasta 1 TB. Los más comunes son de 16 GB, 32 GB y 64 GB.

¿Qué es un USB booteable?

Un USB booteable es un dispositivo que contiene un sistema operativo o una herramienta de arranque que permite iniciar la computadora desde él, en lugar del disco duro interno. Se usa para:

  • Instalar un sistema operativo desde cero.
  • Ejecutar sistemas operativos en modo Live (sin instalar).
  • Recuperar datos o reparar el sistema.
  • Probar distribuciones de Linux.

Para crear un USB booteable necesitas:

  1. Un USB con suficiente capacidad (8 GB o más).
  2. Una imagen ISO del sistema operativo o herramienta.
  3. Un programa para grabar la ISO en el USB (Rufus, Ventoy, Etcher, etc.).

¿Qué es una imagen ISO?

Una imagen ISO es un archivo que contiene una copia exacta de un disco (CD, DVD o Blu-ray). Incluye todos los archivos, carpetas y estructura de arranque necesarios para instalar un sistema operativo o ejecutar una herramienta.

Ejemplos de ISO:

  • Windows 10/11 (descargable desde Microsoft).
  • Distribuciones de Linux (Ubuntu, Fedora, Linux Mint).
  • Herramientas de recuperación (Hiren's BootCD, GParted).
Las ISO suelen ser grandes (2-6 GB) y deben grabarse correctamente en el USB para que sean booteables.

¿Qué es un Sistema Operativo?

Un Sistema Operativo (SO) es el software principal que gestiona el hardware y permite que otros programas funcionen. Es el puente entre el usuario y la máquina.

Windows

El más usado en entornos educativos y empresariales. Fácil de usar, compatible con muchos programas.

Linux

Gratuito, seguro y altamente personalizable. Ideal para servidores y desarrollo, pero con curva de aprendizaje.

Ambos SO pueden instalarse desde un USB booteable.

Herramientas para crear USB booteables

Rufus

Rufus es una herramienta ligera y rápida para crear USBs booteables a partir de imágenes ISO. Es muy popular por su sencillez y eficacia.

Ventajas
  • ✓ Muy rápido (graba la ISO directamente).
  • ✓ Soporta MBR y GPT.
  • ✓ Permite elegir el sistema de archivos (FAT32, NTFS).
  • ✓ Portable (no requiere instalación).
Desventajas
  • ✗ Solo graba una ISO a la vez.
  • ✗ Hay que formatear el USB cada vez que se cambia la ISO.
Descargar Rufus

Ventoy

Ventoy es una herramienta innovadora que permite tener múltiples ISOs en un mismo USB y elegir cuál arrancar desde un menú al inicio.

Ventajas
  • ✓ Soporta múltiples ISOs en un solo USB.
  • ✓ No requiere formatear el USB para agregar/quitar ISOs (solo copiar/pegar).
  • ✓ Compatible con Windows, Linux, herramientas de recuperación.
  • ✓ Fácil de usar, menú gráfico al arrancar.
Desventajas
  • ✗ Ocupa algo de espacio en el USB para su propio sistema.
  • ✗ En algunos equipos antiguos puede tener problemas de arranque.
Descargar Ventoy
Recomendación: si necesitas un solo SO, usa Rufus. Si pruebas varias distribuciones o herramientas, Ventoy es ideal.

Acceder a la BIOS y configurar el arranque

Para arrancar desde un USB, debes entrar a la BIOS/UEFI de tu computadora y cambiar el orden de arranque o seleccionar el dispositivo.

  1. Reinicia la computadora.
  2. Presiona la tecla correspondiente durante el inicio (varía según la marca): F2, Del, Esc, F10, F12 (consulta la pantalla de inicio).
  3. Dentro de la BIOS, busca la sección Boot (Arranque).
  4. Selecciona el USB como primera opción de arranque o usa la tecla de acceso rápido para elegir el dispositivo de arranque (ej. F12).
  5. Guarda los cambios y reinicia.

Aquí tienes un recorrido visual del proceso:

Paso 1: Acceso a la BIOS

Presiona la tecla indicada (F2, Del, etc.) durante el arranque para ingresar a la BIOS/UEFI.

Paso 2: Interfaz de la BIOS

Una vez dentro, verás la pantalla principal de la BIOS. Busca la sección de arranque (Boot).

Paso 3: Sección Boot

En la sección Boot, selecciona el USB como primer dispositivo de arranque. Guarda los cambios (F10) y reinicia.
